## 🎁 安装

> 使用 Github Action 自动打包，无需担心安全问题


- Windows

  > 全功能支持

- Linux

  > 开机自启功能可能会失效

- Mac

  > 由于没有 Apple 开发者账号，打开时如出现 `app 已损坏`，请执行如下指令

  ```bash
  sudo xattr -rd com.apple.quarantine /Applications/Monit.app
  ```
